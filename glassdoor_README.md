# Glassdoor Salary Prediction ML Project

## Overview
This project focuses on analyzing tech job salaries using Machine Learning. The dataset was collected from Glassdoor job postings and includes information such as job title, company rating, company size, industry, revenue, location, and salary estimates.

The main objective of this project is to:
- Analyze factors affecting salaries in the tech industry
- Perform Exploratory Data Analysis (EDA)
- Apply statistical hypothesis testing
- Build and compare Machine Learning regression models
- Predict average salary based on job-related features

---

8. Model Evaluation and Comparison

---

## Key Insights
- Senior-level jobs generally offer higher salaries.
- California-based jobs tend to pay more than jobs in other states.
- Job role and location strongly influence salary prediction.
- Company ratings have only a weak relationship with salary.
- Random Forest Regressor achieved the best model performance.

---

## Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

---

## Best Model
Random Forest Regressor performed the best with:
- Highest R² Score
- Lowest MAE
- Lowest RMSE

---

## Conclusion
This project demonstrates the complete end-to-end Machine Learning workflow, including data analysis, preprocessing, feature engineering, statistical validation, model training, and evaluation.

The project highlights how factors like seniority, job role, and location significantly influence salaries in the tech industry.
